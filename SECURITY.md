# Security Policy

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email **security@ajosave.app** with:
1. Description of the vulnerability
2. Steps to reproduce
3. Potential impact

We will acknowledge within 48 hours and aim to patch critical issues within 14 days.

## Scope

In scope: smart contract vulnerabilities, auth bypass, fund loss, API injection.  
Out of scope: social engineering, DoS, issues requiring physical device access.
